# WeatherApp

A modern, responsive weather forecasting application built with React 19 and Tailwind CSS. This app provides real-time weather updates with a sleek user interface, currently configured for seamless deployment via GitHub Pages.

# Features
- Real-time Weather Data: Integrated to fetch current conditions and forecasts.

- Responsive Design: Built with Tailwind CSS to ensure a great experience on mobile, tablet, and desktop.

- Fast Performance: Leverages React 19 and Web Vitals for optimal loading and rendering.

- Easy Deployment: Pre-configured with gh-pages for one-command publishing.

# Tech Stack
- Frontend: React.js (v19.0.0)

- Styling: Tailwind CSS, PostCSS, Autoprefixer

- Deployment: GitHub Pages

- Build Tool: Create React App (react-scripts)

# Installation & Setup

### Clone the repository:
git clone https://github.com/thinakaran123/react-weather-app.git
cd react-weather-app

### Install dependencies:
npm install

### Start the development server:
npm start

# Available Scripts

- npm start: Runs the app in development mode.

- npm run build: Bundles the app for production in the build folder.

- npm run deploy: Automatically builds and pushes the project to the gh-pages branch.
